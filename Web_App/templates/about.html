About the Audio Classifier Project

This project is designed to classify audio files into three categories: music, speech, and noise. It uses machine learning techniques to analyze the audio data and predict the category with a certain probability.

The project is built using Python, TensorFlow, and Flask for the backend, and HTML, CSS, and JavaScript for the frontend. The model is trained on a dataset of audio files and can accurately classify new audio files uploaded by the user.

The audio classification model is based on MFCC (Mel Frequency Cepstral Coefficients) features extracted from the audio files. These features are commonly used in audio processing and are effective in distinguishing between different types of sounds.

You can view the source code of this project on Google Colab.
